\section{Restricciones Generales}

\begin{itemize}
    \item La red debe ser ingresada como entrada al programa a través del archivo de configuración de red, cuya extensión debe ser inp.
    \item La herramienta solo está disponible para el sistema operativo Window de 64bits.
    \item El sistema debe ser ejecutado con Java 1.8 de 64bits.
\end{itemize}
